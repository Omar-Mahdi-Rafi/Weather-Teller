# Weather-Teller

## Getting Started

First step is to open CLI and set the directory to the project folder location on local drive

Server `Path: \Weather-Teller>`

Run the following commands in CLI
`$ npm init`
`$ npm install`
`$ npm install express`
`Path: \Weather-Teller>node src/app.js`
